# System Map

OpenExtKit is organized as a monorepo with apps, packages, examples, and project documentation.

- `apps/docs`: future VitePress documentation app.
- `packages/core`: configuration and project resolution.
- `packages/manifest`: browser manifest generation.
- `packages/browser`: extension API compatibility wrapper.
- `packages/testing`: browser smoke, visual, visual regression, and dev-session utilities.
- `packages/packaging`: build and package outputs.
- `packages/release`: publish checks, store metadata, and store readiness scoring.
- `packages/mcp-server`: MCP server integration.
- `packages/templates`: project templates.
- `examples`: reference extensions.
