# Installation

OpenExtKit is designed for Node.js 20 or newer and pnpm workspaces.

Inside this repository:

```sh
pnpm install
pnpm build
pnpm test
```

For generated projects, add the packages you need:

```sh
pnpm add -D @openextkit/cli @openextkit/core
```

Use the `openext` binary for project generation, builds, tests, packaging, reports, and MCP startup.
